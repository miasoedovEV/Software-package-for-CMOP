1.1 Структура главных страниц программы

Поскольку выполнение расчета магистрального трубопровода, расстановка станций, перерасчёт стенок трубопровода и расчёт режимов – это последовательные процессы, основные действия с программой реализованы в три этапа.

  • список используемых магистральных насосов;

1.2 Структура меню бара

Пользователю доступны 3 вкладки в меню баре. Вкладка под названием «Меню» включает следующие события:

  • Новый вариант;
  • Сохранить вариант;
  • Удалить все варианты;
  • Экспорт в "xls";
  • Справка.

Вкладка «Вариант» при нажатии левой кнопки мыши открывает список вариантов.

При наведении курсора мыши на один из элементов данного списка откроется перечень действий, включающий: «Открыть», «Удалить».

Вкладка «Внести в базу оборудование» включает следующие события:

  • Добавить магистральный насос;
  • Добавить подпорный насос;
  • Добавить трубу;
  • Добавить коэффициент.

2 Работа со страницами

2.1 Работа с первой страницей

2.1.1 Открытие и закритие существующего варианта

Согласно описанию первой страницы (1.1), пользователю при открытии программы доступен только расчёт магистрального трубопровода, поскольку изначально приложение запускается в режиме «Новый вариант».

В случае, если в базе присутствуют посчитанные варианты, пользователь может открыть необходимый вариант.

Для этого нужно кликнуть правой кнопкой мыши на вкладку на меню баре с надписью «Вариант». Далее откроется список посчитанных вариантов, если таковые имеются. При наведении курсора мыши на нужный вариант откроются два действия: «Открыть» и «Удалить». При нажатии правой кнопки мыши на вкладку «Открыть», просчитанный вариант откроется в программе.

Чтобы начать новый расчёт при открытом варианте, пользователь может нажать кнопку «Сбросить» или открыть вкладку в меню баре под названием «Меню» и нажать «Новый вариант». В первом случае все данные в таблицах ввода сотрутся, во втором случае данные останутся.

2.1.2 Описание расчёта нового варианта

Для начала расчёта нового варианта необходимо заполнить таблицы: Характеристика трассы, Свойства нефти, Координаты трассы, далее нажать кнопку «Рассчитать». Стоит отметить, что данные могут быть целыми и вещественными числами. В случае ввода неверного типа данных откроется окно предупреждения об ошибке.

Для добавления строки в таблицу «Координаты трассы» следует нажать кнопку «Добавить». Если необходимо добавить несколько строк, следует ввести количество строк в поле ввода рядом с кнопкой и нажать на кнопку «Добавить».

Если пользователь верно ввёл данные откроется «Окно ввода магистрального насоса». На данном этапе есть два варианта продолжения расчёта. Пользователь может предоставить выбор магистрального насоса программе из присутствующих в базе данных и продолжить расчёт, нажав кнопку «Пропустить», или ввести свою марку насоса с необходимыми характеристиками.

Как показано на рисунке 7, поле ввода марки магистрального насоса и диаметра дискарабочего колеса принимает строковые значения (любые символы), остальные три поля принимают только целые и вещественные числа.

В случае неверного заполнения данных, пользователь может очистить поля ввода, нажав кнопку «Сбросить».

После верного заполнения данных для продолжения расчёта с пользовательским магистральным насосом, необходимо нажать кнопку «Ввод». Следует отметить, что введённый магистральный насос не вносится в базу данных.

2.2 Работа со второй страницей

В начале работы со второй страницей программа выводит расчётные данные в таблицу «Расчётные данные», с которыми будет производится расчёт толщин стенок трубопровода и максимально допустимых напоров по длине трубопроводов.

У пользователя есть выбор: либо рассчитать только максимально допустимые напоры по длине трубопровода с имеющейся толщиной стенки, либо рассчитать максимально допустимые напоры по длине трубопровода с изменёнными толщинами стенок на участках.

Для расчёта максимально допустимых напоров по длине трубопровода с имеющейся толщиной стенки достаточно заполнить таблицу «Уточнённые категории участков и коэффициенты условий работы» и нажать кнопку «Рассчитать» с подписью «Рассчитать без изменения толщин стенок трубопровода». В результате произведётся расчёт и заполнится таблица «Пересчёт максимально допустимых напоров на участках трубопровода».

Кнопка «Добавить» под таблицей «Уточнённые категории участков и коэффициенты условий работы» добавляет строку в данную таблицу. Если необходимо добавить одну строку достаточно нажать на кнопку «Добавить». Если необходимо добавить несколько строк, следует ввести количество строк в поле ввода рядом с кнопкой «Добавить», и далее нажать кнопку «Добавить».

Кнопка «Ввод» вносит данные из таблицы «Уточнённые категории участков и коэффициенты условий работы» в базу данных по текущему варианту.

Следует указать, что первый столбец принимает строковые значения, второй столбец принимает только вещественные и целые числа, третий столбец принимает целые и вещественные числа.

Для расчёта максимально допустимых напоров по длине трубопровода изменёнными толщинами стенок на участках первым шагом следует внести данные из таблицы «Уточнённые категории участков и коэффициенты условий работы». Для этого необходимо заполнить данную таблицу и нажать кнопку «Ввод».

В результате выполнения данного действия в таблице «Изменённые толщины стенок трубопровода на участках» заполнится первый столбец.

Далее необходимо заполнить второй столбец данной таблицы и нажать кнопку «Рассчитать» с подписью «Рассчитать с изменёнными толщинами».

В итоге таблица «Изменённые толщины стенок трубопровода на участках» заполнится согласно введённым толщинам стенок трубопровода.

2.3 Работа с третьей страницей

Для начала работы на третьей странице необходимо заполнить поле ввода «Разрешённое значение подпора» и «а'». Необходимо отметить, что поля ввода принимают только вещественные и целые числа. После заполнения следует нажать кнопку «Рассчитать».

После выполнения выше перечисленных действий заполнится таблица результатов «Режимы эксплуатации трубопровода для первого эксплуатационного участка».

Кнопка «Сбросить» очищает поля ввода данных.

Необходимо отметить, что при изменении результатов расчёта второй страницы, результаты расчёта третьей страницы аннулируются и таблица «Режимы эксплуатации трубопровода для первого эксплуатационного участка» очищается.

3 Работа с вариантами

Сохранение варианта осуществляется в конце расчёта любой из трех страниц приложения.

Для сохранения варианта следует нажать левой кнопкой мыши на вкладку в меню баре под названием «Меню», далее нажать левой кнопкой мыши на вкладку «Сохранить вариант».

После этого появится окно сохранения. Визуальное представление окна сохранения показано на рисунке 17.

Имена вариантов могут быть любыми. В случае ввода существующего имени, записи старого варианта в базе данных перепишутся на итоги расчёта нового варианта.

Вкладка в меню баре «Вариант» содержит список вариантов, занесённых в базу данных.

Для открытия варианта необходимо нажать правой кнопкой мыши на вкладку в меню баре под названием «Вариант». Далее откроется список существующих вариантов. При наведении курсора мыши на нужный вариант всплывёт список из двух действий: «Открыть», «Удалить».

Чтобы выполнить одно из перечисленных действий, следует нажать левой кнопкой мыши на нужную вкладку.

Начать расчёт нового варианта можно двумя способами:

  • Нажать правой кнопкой мыши на вкладку в меню баре под названием «Меню». Далее нажать правой кнопкой мыши на вкладку «Новый вариант». После этого программа перейдет в режим «Нового варианта». Следует отметить, что при использовании этого метода таблицы ввода данных на первой странице не очищаются.
  • Нажать на кнопку «Сбросить» на первой странице. В следствие этого программа перейдёт в режим «Нового варианта» и таблицы ввода данных очистятся;
  • Изменить какое-либо исходное значение в таблицах ввода данных на первой странице и нажать кнопку «Рассчитать». Программа перейдет в режим нового варианта и начнёт расчёт.

4 Работа со списками оборудования и списком коэффициентов

Для того, чтобы открыть список магистральных насосов, подпорных насосов, труб или коэффициентов следует нажать левой кнопкой мыши на вкладку «Внести в базу оборудование, коэффициент» на меню баре. Далее появится список действий, нажав на которые левой кнопкой мыши откроется окно с нужным списком.

4.1 Работа с окнами списков оборудования

Окна изменения списков магистральных насосов, подпорных насосов, труб работают одинаково.

В качестве примера рассмотрим функционал окна со списком магистральных насосов.

Окно имеет следующие виджеты:

  • Таблица ввода данных («Добавление магистральных насосов»);
  • Кнопки: «Отмена», «Добавить», «Сбросить», «Ввод»;
  • Таблица отображения данных базы.

5 Описание элементов «Меню»

Вкладка «Меню» в меню баре содержит следующие элементы:

  • «Новый вариант»;
  • «Сохранить вариант»;
  • «Удалить все варианты»;
  • «Экспорт в «xls»»;
  • «Справка».

При нажатии на вкладку «Новый вариант» программа переходит в режим нового варианта.

При нажатии вкладки «Сохранить вариант» происходит сохранение расчётов варианта. Более подробно процесс описан в 3 разделе.

При нажатии вкладки «Удалить все варианты» удаляются все варианты из базы данных.

При нажатии вкладки «Экспорт в «xls»» начинается процесс сохранения результатов в файл «xls».

При нажатии вкладки «Справка» открывается окно с руководством по работе с программой.
